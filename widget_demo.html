🌞 Widget d'Alerte Solaire

Surveillance en temps réel de l'activité solaire pour votre site web

📋 À propos

Le Widget d'Alerte Solaire est un composant JavaScript autonome qui permet d'afficher des alertes en temps réel concernant l'activité solaire anormale détectée par les satellites GOES de la NOAA.

⚡ Temps réel

Vérification automatique toutes les 5 minutes

🎨 Personnalisable

Position, intervalle, et comportement configurables

📱 Responsive

Adapté aux ordinateurs, tablettes et mobiles

🔔 Notifications

Pop-up automatique lors d'alertes critiques

🚀 Installation rapide

Étape 1 : Incluez le script du widget dans votre page HTML :

<script src="solar_alert_widget.js"></script>

Étape 2 : Initialisez le widget :

<script> SolarAlertWidget.init({ apiUrl: 'http://localhost:5000', checkInterval: 300000, // 5 minutes position: 'top-right', autoShow: true, showNormalStatus: false }); </script>
💡 Note : Le widget s'affiche automatiquement dans le coin de votre page. Regardez en haut à droite de cette page pour voir le widget en action !

⚙️ Configuration

Option Type Défaut Description
apiUrl String 'http://localhost:5000' URL de l'API d'alertes solaires
checkInterval Number 300000 Intervalle de vérification en millisecondes
position String 'top-right' Position du badge (top-right, top-left, bottom-right, bottom-left)
autoShow Boolean true Afficher automatiquement le pop-up lors d'une nouvelle alerte
showNormalStatus Boolean false Afficher le badge même quand l'activité est normale

📊 Niveaux d'alerte

S1 - Mineur (≥10 pfu) : Risque biologique mineur pour les astronautes en EVA
S2 - Modéré (≥100 pfu) : Risque accru pour les passagers d'avions à haute altitude
S3 - Fort (≥1000 pfu) : Perturbations radio HF dans les régions polaires
S4 - Sévère (≥10000 pfu) : Risques importants pour les vols polaires et satellites
S5 - Extrême (≥100000 pfu) : ⚠️ URGENCE - Risques critiques pour tous systèmes spatiaux
📖 Source : Échelle de tempête de radiation solaire de la NOAA (plus d'infos)

🔧 Prérequis

⚠️ Important : Le widget nécessite que l'API d'alertes solaires soit en cours d'exécution.

Pour lancer l'API sur votre serveur :

# Windows PowerShell .\start_api_server.ps1 # Ou directement avec Python python scripts/solar_alert_api.py

L'API sera accessible sur http://localhost:5000 par défaut.

🌐 Exemple d'intégration complète

<!DOCTYPE html> <html> <head> <title>Mon Site avec Alertes Solaires</title> </head> <body> <h1>Bienvenue sur mon site</h1> <p>Contenu de votre site...</p> <!-- Widget d'alerte solaire --> <script src="solar_alert_widget.js"></script> <script> SolarAlertWidget.init({ apiUrl: 'https://votre-serveur.com:5000', checkInterval: 300000, position: 'bottom-right', autoShow: true }); </script> </body> </html>
